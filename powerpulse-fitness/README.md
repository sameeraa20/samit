# PowerPulse Fitness Website

A modern, fully responsive website for PowerPulse Fitness gym built with React and Tailwind CSS.

## 🌟 Features

- **Fully Responsive Design**: Optimized for all devices (mobile, tablet, desktop)
- **Modern UI/UX**: Clean design with gym-inspired color scheme (black/red/white)
- **Smooth Navigation**: Sticky navbar with smooth scroll functionality
- **Interactive Elements**: Hover effects, animations, and transitions
- **Contact Integration**: WhatsApp floating chat button and contact form
- **Location Integration**: Google Maps embed for gym location

## 🏠 Sections

1. **Hero Section**: Eye-catching banner with gym tagline and call-to-action
2. **Membership Plans**: Card layout displaying all pricing tiers
3. **Features**: Grid layout showcasing gym amenities and facilities
4. **Location**: Address details with integrated Google Maps
5. **Contact**: Contact form and information with WhatsApp integration

## 🚀 Getting Started

### Prerequisites

- Node.js (v14 or higher)
- npm or yarn

### Installation

1. Clone the repository
```bash
git clone <repository-url>
cd powerpulse-fitness
```

2. Install dependencies
```bash
npm install
```

3. Start the development server
```bash
npm start
```

4. Open [http://localhost:3000](http://localhost:3000) to view it in the browser

### Build for Production

```bash
npm run build
```

## 🔧 Customization

To customize the website:

1. **Colors**: Edit the color scheme in `tailwind.config.js`
2. **Content**: Modify text content in `src/App.js`
3. **Styling**: Add custom styles in `src/index.css`
4. **Images**: Replace hero background in the gradient-bg class
